# Campus Travel Connect

Campus Travel Connect is a web-based platform designed for college students to plan and organize group travel efficiently within and beyond campus.

## ✨ Features

- 🔍 Find travel partners for short trips  
- 🧳 Create or join long-distance travel groups  
- 💬 Real-time messaging between users  
- 👤 User authentication & profile management  
- 📊 Dashboard to manage trips and requests  

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas
